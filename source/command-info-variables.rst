================
 Info-Variables
================

.. program:: info-variables

Synopsis
========

:samp:`build-generator {[GLOBAL-OPTIONS]} info-variables [COMMAND-OPTIONS]`

Description
===========

Print information about recognized variables.

.. option:: --filter=REGEX

   * type:``(OR NULL STRING FUNCTION)``

   * default:``false``

   Restrict output to matching variables.
