.. _command-help:

======
 Help
======

.. program:: help

Synopsis
========

:samp:`build-generator {[GLOBAL-OPTIONS]} help [COMMAND-NAME]`

Description
===========

Print help either for all commands or for a given command.

.. option:: [COMMAND-NAME]

   Type: ``(OR NULL STRING)`` Default: ``false``

   Nil or a command name for which to print help.
